# Spielanleitung – Tower of Hanoi

## Ziel des Spiels
Alle 4 Scheiben sollen von einem Startstab auf einen anderen Stab verschoben werden und von der größten zur kleinsten aufgebaut werden.

## Spielaufbau
- 3 Stäbe
- 4 unterschiedlich große Scheiben
- Start: alle Scheiben liegen geordnet vom größten zum kleinsten auf einem Stab 
(größte unten, kleinste oben)

## Regeln
1. Es darf nur **eine Scheibe gleichzeitig** bewegt werden.
2. Es darf nur die **oberste Scheibe** genommen werden.
3. Eine **größere Scheibe** darf niemals auf einer kleineren liegen.



## Spielende
Das Spiel ist beendet, wenn alle Scheiben korrekt auf dem Zielstab aufgebaut sind.
Korrekt ist es wenn die größte scheiben unten ist und dann immer eine kleinere draufliegt bis zur kleinsten ganz oben. Ziel ist so wenig wie möglich Züge zu verwenden.
Man kann wenn das Spiel vorbei ist ein neues spiel starten mit dem reset button.

##Steuerung
Wenn man auf dem button unter einem start drückt und dann auf den anderen bewegt sich die oberste disk auf den angeklickten stab. Wenn man eine Scheibe bewegt zählt es als Zug und und der Counter steigt.
